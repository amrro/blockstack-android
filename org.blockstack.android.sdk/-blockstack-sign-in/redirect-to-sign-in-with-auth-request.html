redirectToSignInWithAuthRequest

suspend fun redirectToSignInWithAuthRequest(context: Context, authRequest: String, blockstackIDHost: String? = null): Unit

Redirects the user to the Blockstack browser to approve the sign in request. To construct a request see the [makeAuthRequest] function.

The user is redirected to the authenticator URL specified in the AppConfig or the default authenticator url

Parameters

authRequest - A request string built by the [makeAuthRequest] function

blockstackIDHost - The ID of the Blockstack Browser application.
