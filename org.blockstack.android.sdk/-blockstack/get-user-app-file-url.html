getUserAppFileUrl

suspend fun getUserAppFileUrl(path: String, username: String, appOrigin: String, zoneFileLookupURL: String?): String

Fetch the public read URL of a user file for the specified app.

Parameters

path - the path to the file to read

username - The Blockstack ID of the user to look up

appOrigin - The app origin

zoneFileLookupURL - The URL to use for zonefile lookup. If false, this will use the blockstack.js's getNameInfo function instead.

Result
the public read URL of the file or null on error
