sign

val sign: Any

(Boolean or String) If set to true the data is signed using ECDSA on SHA256 hashes with the user's app private key. If a string is specified, it is used as the private key instead of the user's app private key. The signature can be verified with GetFileOptions.verify set to true when retrieving the data.
